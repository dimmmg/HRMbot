# Otdel Task & Order Bot

Система управления заказами и задачами агентства Otdel. Бот предоставляет интерфейс оформления заказа услуг, а также внутренний контур для сотрудников (управление задачами, просмотр заработка), интегрируясь с Google Sheets в качестве базы данных.

## Спецификация (Доменная модель)
- **Актори**: Пользователи (физ/юр лица), Сотрудники, Администраторы.
- **Сущности**: Заказ (Корзина услуг, Форма работы), Задача (Дедлайн, Статус, Проект), Услуга (Прайс-лист).
- **Сценарии использования**:
  - Сборка корзины услуг и расчет итоговой стоимости.
  - Подтверждение заказа с автоматической генерацией сметы в таблицах.
  - Назначение задач сотрудникам (только для Администраторов).
  - Отметка задач как "Завершено" и просмотр статистики по заработку.
